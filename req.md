# Requirements for Mighty Empires

## Map

### Tiles and terrain

- Terrain types: 
  - lowland
    - no mountains
  - highland
    - yes river sources
  - river
  - costal

- biomes:
  - plain
  - low hills
  - forest
  - mountains
  - swamp
  - lake
  - river sources
  - river end
  - coast

- tile edges: every edge has a biome
  - mountain -> highland
  - hill -> highland or lowland
  - plain -> lowland
  - forest -> lowland
  - river -> river valley
  - coast -> coastal
  - see -> none
  - swamp -> lowland or river valley
